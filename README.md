# 深圳积分入学助手 - 微信小程序

帮助深圳家长快速计算幼升小入学积分，评估目标学校录取概率。

## 功能特性

- **四区支持**：福田、罗湖、南山、宝安，各区独立积分规则
- **分步问答**：通过简单问答自动计算积分，支持回退修改
- **概率评估**：基于近3年录取分数线，综合趋势分析给出录取概率
- **学校详情**：历年分数线趋势图、积分对比、录取建议
- **本地缓存**：自动保存上次计算结果

## 使用方法

1. 使用 **微信开发者工具** 打开 `sz-score-miniapp` 目录
2. 如果没有 AppID，选择"测试号"即可预览
3. 编译运行

## 项目结构

```
sz-score-miniapp/
├── pages/
│   ├── index/            # 首页 - 区域选择
│   ├── calculator/       # 积分计算器 - 分步问答
│   ├── result/           # 计算结果展示
│   └── school-detail/    # 学校详情与概率分析
├── components/
│   ├── question-card/    # 问题卡片组件
│   ├── progress-bar/     # 进度条组件
│   ├── score-gauge/      # 分数仪表盘组件
│   └── probability-tag/  # 概率标签组件
├── data/districts/       # 各区积分规则与学校数据
├── utils/                # 积分计算引擎与概率算法

```

## 积分规则说明

| 区域 | 分类体系 | 基础分范围 | 特点 |
|------|---------|-----------|------|
| 福田区 | 6类 | 60~80分 | 居住+社保加分，各上限10分 |
| 罗湖区 | ABCDEF | 60~100分 | 类别优先+积分排序，月积分累加 |
| 南山区 | 户籍+住房+社保 | 满分80+19 | 三维度基础分+两项加分 |
| 宝安区 | 8类 | 60~105分 | 加分不封顶 |

## 数据说明

- 积分规则基于2025年各区教育局发布的招生政策
- 学校录取分数线基于2023-2025年公开数据
- 概率评估仅供参考，不代表实际录取结果

## 扩展

如需添加新的区域，在 `data/districts/` 下新建对应 JS 文件，按照现有格式定义 `questions`、`calcScore` 和 `schools`，然后在 `index.js` 中注册即可。
