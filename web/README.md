# SZ Score Web 版

`web/` 目录已提供可运行的 H5 预览工程（Vite + React），并直接复用了主项目的区级规则与积分计算逻辑：

- 数据来源：`../data/districts/*`
- 计算引擎：`../utils/score-engine.js`

## 本地启动

```bash
cd web
npm install
npm run dev
```

默认访问：`http://localhost:5173`

## 当前能力

1. 选择区级与学段（幼升小/小升初）。
2. 按规则动态展示问题（含条件题）。
3. 实时计算总分、类别与加分明细。
4. 根据片区筛选并展示学校列表（示例展示前 10 所）。

## 后续可扩展

1. 拆分为首页/计算页/结果页/学校详情页多路由结构。
2. 增加历年分数趋势图与概率可视化。
3. 与小程序 UI 组件风格进一步对齐。
